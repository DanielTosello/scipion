Step 1: Upload and import your files to the web service clicking in the graph at left on the box Import Averages.



Step 2: Select a protocol to use with your averages. You can choose between RANSAC, EMAN initial volume, Significant and Prime.



The parameters can be customized to get the best results. If you need more options, just change the Expert Level to show the advanced options.



Finally press "Execute"!


Step 3: When the protocol is finished you can see a summary about the protocol at the bottom.



Then you can visualize the objects by clicking over the names or using the Analyze Results button.




Step 4: After execute more runs you can join and align the volumes generated to visualize all the set using the align volumes box.



Step 5: You can download your results using the button at the bottom of the page after mark or select an execution.
