My First Map

Create your initial volume


This webpage will allow you to create an initial 3D map from your average images. Several methods from different packages are available (Xmipp3, Eman2, ...). At the end, you compare the output volumes and download the desired ones.

It is recommended to check and read the tutorial before to use this tool.

First Step: Your project

Create a new project (or go to a previous one)



Second Step: Using data

Upload binary image stack file and import this averages into the Project.

Final Step: Running

Run different initial volumes methods. After you can compare and download the results.
